# pkb_study_analysis_TC
R scripts for the statistical analysis and power calculation of a study on Parkinson´s disease complex treatment (PKB).

This repository contains the R code used for the statistical analyses of the dissertation:

"Multimodale Evaluation der Parkinson-Komplex-Behandlung: Auswirkungen auf Motorik und Lebensqualität von Patienten mit Parkinson-Krankheit"

## Contents

- `01_power_analysis.R`: Sample size calculations and power analysis
- `02_led&baseline_calculation.R`: Baseline and LED Calculation
- `03_TUG_analysis.R`: TUG Dataimport and main analysis
- `04_UPDRS_analysis.R`: UPDRS Dataimport and main analysis
- `05_PDQ39_analysis.R`: PDQ39 Dataimport and main analysis
- `06_PKBE_analysis.R`: PKB-E Dataimport and main analysis
- `07_plots_main_results.R`: Visualization of Main Results via Scatterplots and grouped Barchart
  
## Software

All analyses were performed using:

- R (R Core Team)
- Version: 4.2.2

## Notes

- The dataset is not publicly available due to data protection regulations.
- The scripts are structured to allow full reproducibility of the reported results given access to the original dataset.

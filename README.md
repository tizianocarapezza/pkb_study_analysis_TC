# pkb_study_analysis_TC
R scripts for the statistical analysis and power calculation of a study on Parkinson´s disease complex treatment (PKB).

This repository contains the R code used for the statistical analyses of the dissertation:

"Effects of Parkinson’s Disease Multimodal Complex Treatment (PKB) on gait performance, motor function, and quality of life"

## Contents

- `01_data_preparation.R`: Data import and preprocessing
- `02_main_analysis.R`: Statistical analyses (TUG, UPDRS III, PDQ-39, subgroup analyses, correlation analyses)
- `03_power_analysis.R`: Sample size calculations and power analysis

## Software

All analyses were performed using:

- R (R Core Team)
- Version: 4.2.2

## Notes

- The dataset is not publicly available due to data protection regulations.
- The scripts are structured to allow full reproducibility of the reported results given access to the original dataset.
